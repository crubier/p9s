# p9s Documentation Website

Documentation website for p9s, built with [Docusaurus](https://docusaurus.io/).

## Development

```bash
bun install
bun start
```

This starts a local development server at http://localhost:3000.

## Build

```bash
bun run build
```

Generates static content into the `build` directory.

## Deployment

```bash
bun run deploy
```

Deploys to GitHub Pages.
